# llm-client

This repository contains the standalone `llm-client` package
(`llm_client` import path).

It is the canonical reusable LLM and agentic runtime package, including:

- provider abstractions
- a unified content and request model
- execution engine, routing, retries, and caching
- agent and tool runtime layers
- observability, replay, validation, and budgets
- cookbook examples and package guides
