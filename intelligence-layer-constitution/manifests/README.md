# Manifests (v1)

This folder contains discovery/configuration artifacts used by the Kernel at startup.

## Intent registry

- `manifests/intent-registry.v1.json`: list of supported intent types and their schema refs.

## Capabilities

- `manifests/capabilities/*.json`: versioned capability manifests (see `9-capabilities.md`).

## Plugins

- `manifests/plugins/operators/*.json`: per-operator plugin manifests (see `5-plugin-contracts.md`).
